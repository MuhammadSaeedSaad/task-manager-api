Routes

User routes

  • POST /users (doesn't require authenticaion)

    • for signing up

  • POST /users/login (doesn't require authenticaion)

    • for logging in

  • POST /users/logout

    • for logging out

  • POST /user/logoutall

    • for logging out from all sessions

  • GET /users/me

    • viewing your profile

  • PATCH /users/me

    • editing your profile

  • DELETE /users/me

    • deleting your account

  • POST /users/me/avatar

    • changing your profile picture

  • GET /users/:id/avatar

    • viewing your profile or any other user's

  • DELETE /users/me/avatar

    • Deleting your profile picture

Task routes

  • POST /tasks

    • creating a task

  • GET /tasks

    • viewing all created tasks

  • GET /tasks/:id

    • viewing a single task by its id

  • PATCH /tasks/:id

    • editing a task

  • DELETE /tasks/:id

    • deleting a task
